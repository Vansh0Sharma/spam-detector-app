# 📩 Spam Message Detector

An AI-powered web app that detects spam messages using a machine learning model trained on the SMS Spam Collection dataset.

## 🔍 Features
- 🧠 Trained ML model (Naive Bayes)
- 🧪 REST API with Flask
- 🌐 Web UI with Streamlit
- 🔁 Real-time prediction
- ☁️ Easy to deploy

## 📦 Tech Stack
- Python, Scikit-learn
- Flask (API)
- Streamlit (Frontend)
- Render.com (Hosting)
